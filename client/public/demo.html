This page demonstrates how easy it is to embed a real-time chat widget on your website. The widget allows visitors to chat with each other, providing a community experience to your site.

How to Embed the Widget

Simply add the following script tag to your HTML:

<script 
  src="https://your-website.com/widget/chat-widget.js" 
  data-chat-widget-auto="true" 
  data-chat-widget-position="bottom-right"
></script>

Configuration Options

You can customize the widget using data attributes:

<script 
  src="https://your-website.com/widget/chat-widget.js" 
  data-chat-widget-auto="true" 
  data-chat-widget-position="bottom-right"
  data-chat-widget-title="Community Chat"
  data-chat-widget-open="false"
  data-chat-widget-width="350px"
  data-chat-widget-height="500px"
></script>

Programmatic Usage

You can also instantiate and control the widget programmatically:

// Initialize the widget
const chatWidget = new ChatWidget({
  position: 'bottom-right',
  title: 'Community Chat',
  initiallyOpen: false,
  width: '350px',
  height: '500px'
});

// Open the widget
chatWidget.open();

// Close the widget
chatWidget.close();

// Toggle the widget
chatWidget.toggle();
