Features of Our Chat Widget

  • URL-Based Chat Rooms: Automatically creates chat rooms based on the current URL, connecting users browsing the same content.
  • Real-Time Messaging: Instant delivery of messages with no page refreshes required.
  • User Presence: See who else is viewing the same page in real-time.
  • Customizable: Easily adjust the widget's appearance, position, and behavior to match your site's design.
  • Lightweight: Minimal impact on your page load times and performance.
  • Easy Integration: Simple script tag or JavaScript API for adding to your site.

How to Implement on Your Site

Adding the chat widget to your website is easy. Just add the following script tag to your HTML:

<script 
  src="https://your-domain.com/widget/chat-widget.js" 
  data-chat-widget-auto="true" 
  data-chat-widget-position="bottom-right"
  data-chat-widget-title="Chat"
  data-chat-widget-open="false"
  data-chat-widget-width="350px"
  data-chat-widget-height="500px"
></script>

Or use the JavaScript API for more control:

// Initialize the chat widget
const chatWidget = new ChatWidget({
  position: 'bottom-right',
  title: 'Chat',
  initiallyOpen: false,
  width: '350px',
  height: '500px'
});

// You can programmatically control the widget
// chatWidget.open();
// chatWidget.close();
// chatWidget.toggle();
